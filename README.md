# HMG-v2 — Harmonogram Výroby

Verze 2 aplikace pro řízení výroby obalovny.

## Nové funkce oproti v1
- [ ] PostgreSQL databáze (persistent data)
- [ ] Přihlašování a role (Admin / Viewer / Guest)
- [ ] Mobilní zobrazení (responzivní)
- [ ] PDF export měsíčního přehledu
- [ ] Graf výroby
- [ ] Historie změn

## Technologie
- Node.js + Express
- PostgreSQL
- HTML / CSS / JS
